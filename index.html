Welcome to cICB Pipeline!

cICB is a computational pipeline for protein annotation. Using an integrative approach, it allows the user to annotate several protein features such as domains, TMHs, disorder and identity through PSIBLAST, HHpred and InterProScan.

System Requirements

In order to run ICB, your system needs the following programs and tools.

  • Any Linux distribution
  • Python 2.7.x
  • Perl 5.x or newer
  • Bash

Due to the modular approach, you need to install one, a few or all the following tools (according to your needs).

-PSIBLAST

-HHsuite

-InterProScan

-PSIPRED

-IUPRED

-TMHMM

The config.txt file

A config.txt file is included in the ICB package. After installing the tools required by ICB, please modify the config.txt file according to your installation paths and the amount of cores available for BLAST and HHblits/HHpred.

Database Requirements

The PSIBLAST Module requires a custom, reduced version of UniProt/SwissProt, available at http://pvcbacteria.org/mywiki/databases/

Input Requirements

The pipeline takes as an input a single or a multiFASTA with the filename ending with "_proteins.fa"

Usage

To run the pipeline, call the main script with 

$ python icb.py -i your_seq_proteins.fa

Use the following flags:

  • -o output folder (default->cwd)
  • -psi PSIBLAST
  • -tmh TMHMM
  • -hhpred HHpred on PDB
  • -hhblits HHblits on UniProtKB
  • -sigp SignalP
  • -iup IUPRED
  • -all Runs all modules
